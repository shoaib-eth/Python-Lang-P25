# Object Types / Data Types

- Numbers : 1234, 3.1415, 3 + 4j, 0b111, Decimals(), Fraction()

- String : 'Hello', "World", '''Multi-line''', f"Formatted {var}"

- List : [1, 2, 3], ['a', 'b', 'c'], list(range(5))

- Tuple : (1, 2), ('x', 'y', 'z'), tuple('abc')

- Dictionary : {'key': 'value'}, dict(a=1, b=2) (Note: In dictionaries, we have not index based access but key based access)

- Sets: set('abc'), {'a', 'b', 'c'}, frozenset([1, 2, 3]), set(range(5))

- Files : open('file.txt', 'r'), with open('file.txt', 'w') as f:

- Boolean : True, False, bool(1), bool(0)

- NoneType : None

- Functions, Modules, Classes

- Advance : Decorators, Generators, Iterators, Meta Programming
